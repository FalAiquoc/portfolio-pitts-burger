# Pitts Burger — Site Institucional
Site desenvolvido como projeto de portfólio por Diogo Falcão (FalAiquoc)

🏪 Sobre o Negócio
O Pitts Burger é uma das maiores redes de hamburguerias artesanais de Natal/RN. Fundada em 1984, destaca-se pelo blend grelhado na hora, ingredientes selecionados e o famoso molho especial Pitts, sendo parada obrigatória em Lagoa Nova.

🛠️ Tecnologias
- Vite + React + TypeScript
- Tailwind CSS
- Lucide React

📐 Referências de Design
- https://github.com/adrianhajdin/project_hoobank — Referência para layout de impacto e design responsivo.
- https://github.com/phuoc-ng/css-layout — Padrões geométricos modernos de CSS e Tailwind para interfaces web.
